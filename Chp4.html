Lesson 04 : Data structures and arrays

Numbers and strings are units and object comprises of set of units

Data set

  • Js provides a sequence of data to store values called arrays
    • Notations is [] for declaration of an array

    Properties

  • Using myString.length to access length of an array

    • Methods

  • every string has toUpperCase() & toLowerCase() property to change casing of letters
  • Properties containing functions are called methods
  • Eg : let sequence = [1, 2, 3];
    sequence.push(4);
    sequence.push(5);
    console.log(sequence);
    // → [1, 2, 3, 4, 5]
    console.log(sequence.pop());
    // → 5
    console.log(sequence);
    // → [1, 2, 3, 4]

  • The push functions add value in end of an array & pop removes last value of an array

    • A stack is a data structure where we push values and pop in opposite order such that things added last is removed first

    Objects

  • Grouping in single values and put it into an array
  • Braces either include a block element or describe an object
  • delete is used to remove named property of an object

    • Mutability

  • It is impossible to change types of values like strings, numbers and booleans
  • But objects work differently, you can change to different contents at different times

    • let object1 = {value: 10};
    let object2 = object1;
    let object3 = {value: 10};
    console.log(object1 == object2);
    // → true
    console.log(object1 == object3);
    // → false
    object1.value = 15;
    console.log(object2.value);
    // → 15
    console.log(object3.value);
    // → 10

  • object1 and object2 has same identity
