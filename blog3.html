Lesson-01 : Values, Types, and Operators

Values

  • A volatile memory has more than 30billions bits
  • bit accumulate a piece of information to form values
  • There are 6 basic types of values; number, Boolean, object, function and undefined
  • Values must have names to be identified and used

Numbers

  • Js uses fixed no. of bits say 64 to store numbers
  • 64 bits can be represented in 2^64 ways
  • Sign and decimal place too occupies a bit each.
  • Also exponential (e) can be used to represent large no. though not precise like pie

    Arithmetic

  • They perform operations with values using operator(*vv
  • The operation is performed by precedence
  • % gives remainder

    Special no.

  • First two are infinity and -infinity#NaN

    Strings

  • Used under single/double quotes
  • \"\\n\"." indicates newline inserted
  • Js uses 16 bits per string element
  • "+" can be used to concatenates strings

    Uniary operators

    - console.log() produces one value - Operators with 2 values are binary operators

    Boolean

    - Used for producing two value "on/off"

    Comparison

    - <(greater than), >(Less than), ==(equals), !=(not equal to) are comparitive operators.

    Logical

    - &&(and), !(not), ||(or) are logical operators

    ternary operators

    - console.log("A" > "a" ? true : false ); also called conditional operator.

    Empty values

    - no meaningful values - null & not-defined

    automatic type conversion

  • convert one type of operator to heigher operator
  • Ex. console.log("5" -1); // -> 4
  • Ex. console.log("5" + 1); //-> 51, this concates string so gives such output
  • console.log(null== undefined); //-> true
  • console.log( null == 0); //-> false
  • console.log(0== false) //->true
  • console.log(""==false) //->true, holds true due to type conversion
